## Blog do Lucas Winter sobre ciencia de dados

Projeto base em `Next.js` para um blog editorial sobre ciencia de dados.

### O que ja existe

- Home com identidade visual e secoes editoriais
- Paginas dinamicas de posts em `/posts/[slug]`
- Conteudos iniciais em `lib/posts.ts`
- Estrutura pronta para evoluir para `MDX`, CMS ou analytics

### Como rodar

1. Instale as dependencias com `npm install`
2. Inicie com `npm run dev`
3. Acesse `http://localhost:3000`
